Projeto de uma API de um CRUD de clientes, utilizando thymeleaf, bootstrap, springboot, e H2 database.

O projeto atua como uma página web destinada para gerência de cadastros de clientes em um banco de dados,
onde é possível criar, excluir e alterar o cadastro de um usuário, além da possibilidade de ver uma listagem de todos os clientes já cadastrados.
O projeto conta com a implementação da chamada de serviço da API do via cep, onde ao preencher o campo com o seu cep, todos os outros dados referentes
ao seu endereço são automaticamente preenchidos pela API.

Para acessar o programa, basta digitar na guia de buscas do seu navegador a URL: http://localhost:8081/clientes/listar

Para ter acesso diretamente ao banco de dados utilizado nesta API, use a URL: http://localhost:8081/h2-console
